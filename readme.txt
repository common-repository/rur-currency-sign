Implements <a href="http://en.wikipedia.org/wiki/Russian_ruble">Russian rubble sign</a> for the [rur] and [p] shortcode in posts.

== Description ==
Upload the RUR Currency Sign plugin to your blog, Activate it, then use [p] or [rur] shortcode after the product price and publish it.
1, 2, 3: You're done!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload all plugin files to the `/wp-content/plugins/rur/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [p] or [rur] shortcode in your content posts
